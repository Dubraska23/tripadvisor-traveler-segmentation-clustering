# tripadvisor-traveler-segmentation-clustering
Segmentación de perfiles de viajeros mediante K-Means y DBSCAN utilizando reseñas de Tripadvisor. Implementa reducción de dimensionalidad PCA y un estudio comparativo de estabilidad de clusters. Enfoque en la gobernanza de metadatos conductuales y gestión de outliers para Business Analytics (UAM).
